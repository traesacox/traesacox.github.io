Typography

The fonts that will be used on the website is Courgette for headings and Kalam for the header and footer element with Candara, Gadget, serif as default fonts for browsers that do not support my inital selections. For body elements the font will be Patrick Hand and cursive will be the default font. I chose these fonts because they are easy to read and yet has a casual, fun feel.
